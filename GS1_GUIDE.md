# Вкладка "Фасовка GS1" - Краткое руководство

## Что добавлено

Новая вкладка для генерации GS1 кодов формата `99MPUC<GS>...` с поддержкой:
- Штучных и весовых товаров
- Автоматической генерации UniqueID при наличии скидки
- Дробных количеств (AI 97)
- QR-кодов + опциональных линейных штрихкодов
- Карусели с автоматической ротацией

## Как использовать

1. **Откройте вкладку "Фасовка GS1"**

2. **Заполните форму:**
   - Название папки (опционально)
   - GoodsId коды (по одному на строку, 1-8 цифр)
   - Выберите тип товара: Штучный / Весовой
   - Настройте количество/вес: Random (диапазон) / Фиксированный
   - Настройте скидку: Фиксированная / Random
   - Укажите количество вариаций на каждый GoodsId
   - При необходимости включите чекбокс "Показать линейный штрихкод"

3. **Нажмите "Добавить"**

4. **Выберите папку и нажмите "Запуск"**

5. **Управление каруселью:**
   - Кнопки ❮ ❯ или стрелки ← → на клавиатуре
   - Кнопка "Стоп" для паузы
   - Настройка интервала ротации (0.5, 0.7, 1, 2 сек)

## Формат GS1

### Штучный товар
```
99MPUC<GS>240[GoodsId]<GS>37[Quantity]<GS>[98[Discount]<GS>21[UniqueID]<GS>][97[DecPos]<GS>]
```

### Весовой товар
```
99MPUC<GS>240[GoodsId]<GS>3103[Weight]<GS>[98[Discount]<GS>21[UniqueID]<GS>]
```

## Примеры

**Штучный, 50 шт, без скидки:**
```
99MPUC<GS>240123<GS>3700000050<GS>
```

**Штучный, 12.45 шт, скидка 10%:**
```
99MPUC<GS>240456789<GS>3700001245<GS>9810<GS>21ABC12345<GS>972<GS>
```

**Весовой, 500г, скидка 5%:**
```
99MPUC<GS>240888777<GS>3103000500<GS>9805<GS>21XYZ67890<GS>
```
